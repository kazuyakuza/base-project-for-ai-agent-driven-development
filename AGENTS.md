# AI Agent Onboarding

This document provides essential information for AI agents working on this project.

## Workflow

All agents must follow the project workflow outlined in the `.ai-agent/WORKFLOWS.md` file. This document details the standardized procedures for development, testing, and deployment.

## Rules

Adherence to the project's coding standards and operational rules is mandatory. These are documented in the `.ai-agent/RULES.md` file.

## Memory Bank

All agents must read the [Memory Bank Instructions](.kilocode/rules/memory-bank/memory-bank-instructions.md) at the beginning of each task. This ensures that the agent has a complete understanding of the project's context, architecture, and technical specifications.
